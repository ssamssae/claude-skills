# 날씨/미세먼지 아침 알림

## 실행 흐름

### 1. 날씨 검색

웹 검색 2회:
- `"서울 오늘 날씨 ${YYYY-MM-DD}"`
- `"서울 미세먼지 초미세먼지 오늘"`

### 2. 정보 추출

추출 항목:
- 현재 기온 / 최저~최고
- 하늘 상태 (맑음/흐림/비/눈)
- 강수 확률
- 미세먼지 (PM10) 수치 + 등급 (좋음/보통/나쁨/매우나쁨)
- 초미세먼지 (PM2.5) 수치 + 등급
- 자외선 지수 (있으면)

### 3. 외출 판단

기준:
- 미세먼지 "나쁨" 이상 → 마스크 권장
- 비 올 확률 50%+ → 우산 챙기세요
- 기온 5도 이하 → 두꺼운 옷
- 기온 30도 이상 → 더위 주의

### 4. 텔레그램 전송

`~/.claude/channels/telegram/send.sh 538806975 "<메시지>"` 로 전송.
(스케줄 잡에서 MCP 텔레그램 플러그인과 충돌 방지를 위해 curl 직접 호출)

포맷:
```
🌤 서울 날씨 (MM/DD 요일)

🌡 기온: 현재 X°C (최저 Y / 최고 Z)
☁️ 하늘: 맑음 | 강수확률 X%
💨 미세먼지: XX㎍/㎥ (보통)
🔬 초미세먼지: XX㎍/㎥ (좋음)

💡 한줄평: [외출 판단 요약]
```

send.sh 실패 시 터미널 출력만.

- 장황하게 쓰지 말 것 — 위 포맷 5줄 이내
